書籍『モデルベース要件定義テクニック』の図版をPlantUMLで描いてみる
====

内容理解のため、神崎善司『モデルベース要件定義テクニック』(2013) ISBN:9784798039442 の図版をPlantUMLで描いてみます。

PlantUMLでは1つの図の中に複数種類のUML(ex. アクティビティ図とユースケース図など)の要素を混在させて描画できないので部分的にnoteとして表現しています。

# 作成方針

- 体裁の深追いはしない
- 意味が合っていれば良しとする
